# Moody Buddy+

A new kind of digital journal for kids aged 7-12. Our mission is to help kids understand their feelings and build happy habits in a fun and private way.

- **Collaborative Storytelling:** Kids can write stories together with their AI "Buddy".
- **Points & Streaks:** Gamified elements to encourage daily engagement.
- **Private Journal:** A safe space for kids to write down their thoughts and reflections.
- **PWA Ready:** The app is a fully functional Progressive Web App that can be installed on any device.
- **Multilingual:** Supports English, Macedonian, and Turkish.

This project is built with React, TypeScript, and Vite. It's ready to be deployed on **Vercel**.

### Prerequisites

- Node.js (LTS version)
- A Google AI API Key

### Local Setup

1.  **Clone the repository.**
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Create an environment file:**
    - Create a file named `.env` in the root of the project.
    - Add your Google AI API key to it. The key **must** be named `VITE_API_KEY`.
      ```
      VITE_API_KEY=YOUR_API_KEY_HERE
      ```
4.  **Run the development server:**
    ```bash
    npm run dev
    ```

### Deployment on Vercel

1.  Push the code to a GitHub, GitLab, or Bitbucket repository.
2.  Go to your Vercel dashboard and create a **New Project**.
3.  Import the repository you just created. Vercel will automatically detect that it is a Vite project and configure the build settings.
4.  Before deploying, expand the **Environment Variables** section.
5.  Add the following environment variable:
    - **Name:** `VITE_API_KEY`
    - **Value:** `YOUR_API_KEY_HERE`
6.  Click **Deploy**. Vercel will build and deploy your app.
